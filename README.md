# teste_tecnico_01

Implemente uma função que receba um número inteiro positivo e retorne o
somatório de todos os valores inteiros divisíveis por 3 ou 5 que sejam inferiores ao
número passado.

* Getting Started

Em seu prompt de comando:
* $ git clone Ctrl + ins
* $ cd teste_tecnico_01
* Abra a IDE
* $ Flutter run
